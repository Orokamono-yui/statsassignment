Question(仮説)

  1. variables being studied -どの値を分析対象にするのか
  2. population groups -どのグループの人々を対象にするのか
  3. population parameter -対象にした人々の属性は何か
  4. direction of the comparison -比較をどの方向性で行うか
  5. Purpose -分析の目的
  6. hypothesis -分析前に立てる仮説

Method(検証方法)

  1. Random samples relevant -関連性あるサンプルをランダムに選んでいるか
  2. what the population is -サンプルは何の集団なのか
  3. sample size -入手できたサンプルの数
  4. sampling method -サンプル抽出の方法

Result(結果)

  1. appropriate graph -適切なグラフ
  2. summary statistics -統計の要旨

Discussion(分析)

  1. comparative feature -グラフ・箱ひげ図の特徴の比較
  2. confidence interval -信頼区間
  3. contextual knowledge -データの意味の知識、PISA・IQは何を測るのかなど

Conclusion(結論)

  1. confidence interval... inference -信頼区間を基にした推論
  2. sample variability -サンプルの散らばり方
  3. context -データが語る背景や実態のこと
  4. reflected, considered -プロセスを反省し、他の説明ができないか考えること
